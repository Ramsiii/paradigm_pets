# Paradigm Pet Professionals

A responsive website redesign for Paradigm Pet Professionals, developed 
as part of the WGU UI Design (D279) course.

View page here: https://ramsiii.github.io/paradigm_pets/home.html

## Pages
- `home.html` — Overview, adoption resources, consultation CTA
- `cats.html` — Cat care by life stage (feeding guide tables)
- `dogs.html` — Dog diet, exercise, grooming, and vaccinations
- `small-pets.html` — Small animal housing and care resources
- `contact.html` — Virtual consultation request form
- `FAQs.html` — Common pet care questions

## Features
- Responsive layout (desktop and mobile via hamburger menu)
- Site-wide keyword search with topic-based navigation (`search.js`)
- Accessible: skip links, ARIA labels, keyboard navigation, WCAG contrast
- SEO meta tags targeting "new pet consultation" keywords
- Client-side form validation on consultation request form
